Mohammed Ibrahim — Portfolio (ملف تعريفي)

صفحة شخصية ثابتة لعرض المهارات والمشاريع وطرق التواصل.

2026 · Front‑End

الملخّص

موقع بسيط ومنظّم يُظهر مقدّم الموقع، مهاراته (HTML, CSS, JS, Tailwind)، وعرض لمشاريعه مع روابط معاينة مباشرة، إضافة إلى نموذج اتصال مع تحقق جهة العميل وميزة وضع الوضع الداكن.

الميزات التقنية

HTML5 CSS3 Tailwind JavaScript Dark Mode (localStorage) Responsive

هيكل المشروع (ملفّات مهمة)

- index.html: الصفحة الرئيسية - about.html: نبذة ومهارات - projects.html: شبكة مشاريع مع روابط Live Demo - contact.html: نموذج اتصال مع تحقق - script.js: منطق الواجهة (Dark Mode، قائمة الموبايل، التحقق) - style.css: أنماط إضافية

كيفية التشغيل السريع

شغّل سيرفر ملفات ثابتة في مجلد المشروع ثم افتح المتصفح على العنوان المحلي (مثال):

python3 -m http.server 8000
# ثم افتح http://localhost:8000

اقتراحات للتحسين

  • ربط نموذج الاتصال بخدمة backend أو Formspree لإرسال رسائل فعلية.
  • تحسين الأداء بضغط الصور واستبدال CDN إن لزم.
  • إضافة اختبار الوصولية (a11y) وتحسين علامات ARIA.
