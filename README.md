# Mohammed Ibrahim — Portfolio

ملف التعريف الشخصي لمطوّر واجهات أمامية يعرض صفحة هاب (Homepage)، نبذة، مشاريع، وصفحة اتصال.

## نظرة عامة
- نوع المشروع: موقع محفظة شخصية (Static portfolio)
- الهدف: عرض المهارات والمشاريع وسهولة التواصل مع صاحب الموقع
- لغة العرض: HTML, CSS, JavaScript مع Tailwind CSS

## الميزات الرئيسية
- صفحة رئيسية (سيرة ذاتية قصيرة ونداء للعمل)
- صفحة About تحتوي على المهارات والاهتمامات
- صفحة Projects تعرض روابط لـ Live Demos لثلاث مشاريع
- صفحة Contact مع نموذج تحقق بسيط من المدخلات
- نمط داكن/فاتح قابل للتخزين في `localStorage`
- أزرار تفاعل: قائمة الهاتف، زر رجوع للأعلى

## التقنيات المستخدمة
- HTML5
- CSS3 + Tailwind CSS
- JavaScript (vanilla)
- أيقونات: Font Awesome

## تشغيل المشروع محليًا
1. افتح مجلد المشروع في الطرفية:

```bash
cd ~/Desktop/portofolio
```

2. أبسط طريقة لتشغيل الموقع (خدمة ملفات ثابتة):

```bash
# بايثون 3
python3 -m http.server 8000
# ثم افتح http://localhost:8000 في المتصفح
```

أو استخدم امتداد Live Server في VSCode أو أي سيرفر ثابت آخر.

## نشر
- يمكن النشر بسهولة على GitHub Pages عن طريق رفع المستودع ثم تفعيل Pages من إعدادات المستودع.

## تخصيص
- تعديل النصوص والصور في `index.html`, `about.html`, `projects.html`, `contact.html`.
- تعديل الألوان والخطوط عبر `style.css` أو إعدادات Tailwind.
- تغيير الروابط الحية (Live Demo) في `projects.html` إلى مشاريعك الفعلية.

## ملاحظات فنية
- التحقق من صحة نموذج الاتصال يتم على مستوى العميل فقط (JS). لإرسال رسائل حقيقية، اربط النموذج بخدمة backend أو Form handling service.
